# 网站图标设计草稿

## 📋 项目信息

- **项目名称：** sniipet-huamn 碎片化人类
- **核心功能：** AI 智能大纲整理工具
- **关键词：** 大纲、树状结构、AI整理、碎片知识、思维连接

---

## 🎨 设计方案

### 方案 A：树形连接图
**概念：** 碎片知识通过连接形成知识树

```
┌─────────────────────────┐
│      ○       ○          │
│     ╱ ╲     ╱           │
│    ○   ○──○            │
│   ╱     ╲               │
│  ○       ○              │
│                          │
│   知识碎片 · 智能连接    │
└─────────────────────────┘
```

**颜色方案：**
- 主色：深蓝色 (#2563EB)
- 强调色：橙色 (#F59E0B)
- 背景：白色渐变

**含义：**
- ○ 节点代表知识碎片
- ╱ ╲ 线条代表 AI 智能连接
- 树形结构代表有序的大纲

---

### 方案 E：AI + 树 (推荐)

- ▲▽▲ AI 芯片符号
- 树形结构代表大纲
- ● 核心节点代表主要思想
- ○ 分支节点代表碎片知识

## 🎯 推荐组合

- ✅ 在国内产品中差异化明显

## 📐 技术规格

### 图标尺寸
- **Favicon:** 32x32px, 16x16px
- **Touch Icon:** 180x180px
- **PWA Icon:** 512x512px
- **推荐源文件:** 1024x1024px (SVG)

### 文件格式
- **主格式:** SVG (矢量，可缩放)
- **备用格式:** PNG (兼容性)
- **Favicon:** ICO (传统格式)

### 颜色模式
- RGB / sRGB
- 支持浅色和深色模式
