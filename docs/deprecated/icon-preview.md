# 图标预览 - 橙色极简方案

## 🎨 最终设计

**方案 C：极简抽象 - 碎片到有序的转化**

### 视觉描述

```
┌─────────────────────────────────────┐
│                                     │
│  ▦▧▩          ⟹        ▦▧▩       │
│ ▧▩▦▧                 ▦▧▩          │
│   ▧▩                ▦▧▧           │
│  ▦▧                 ▦             │
│                     ▦▧▩           │
│                                     │
│  碎片          ⟹      整理大纲     │
│ (杂乱)              (有序)         │
└─────────────────────────────────────┘

颜色：橙色渐变系
```

---

## 🎨 设计细节

### 左侧：杂乱的碎片
- 6 个大小不一的方块，错落排列
- 代表：零散的知识碎片
- 颜色：橙色系不同透明度（0.7-0.9）
- 传达：无序、混乱、碎片化

### 中间：处理箭头
- 简洁的箭头符号
- 代表：AI 智能处理过程
- 颜色：深橙色 (#9A3412)
- 传达：转化、处理

### 右侧：整齐的大纲
- 层级分明的矩形块
- 代表：有序的大纲结构
- 颜色：橙色渐变（从深到浅）
- 传达：有序、清晰、结构化

---

## 🌈 颜色方案

### 主色调
```css
/* 背景色 */
--bg-light: #FFF7ED;  /* 浅米白 */

/* 橙色渐变系 */
--orange-dark:  #EA580C;  /* 主色 */
--orange-main:  #F97316;  /* 标准 */
--orange-light: #FB923C;  /* 浅色 */
--orange-pale:  #FDBA74;  /* 极浅 */
--orange-deep:  #9A3412;  /* 深色 */
```

### 使用场景
- **图标主体：** #EA580C（深橙）
- **高光部分：** #FDBA74（浅橙）
- **背景：** #FFF7ED（米白）
- **文字/线条：** #7C2D12（深棕）

## 📐 技术规格

| 属性 | 值 |
|------|-----|
| 文件格式 | SVG |
| 尺寸 | 512×512px |
| 视口 | 0 0 512 512 |
| 圆角 | 100px (外框) |
| 方块圆角 | 5-8px |
| 文件大小 | ~2KB |

### 响应式
SVG 是矢量格式，可以无损缩放到任何尺寸：
- 16×16 (favicon)
- 32×32 (favicon)
- 180×180 (Apple Touch)
- 512×512 (PWA)

## ✨ 设计亮点

1. **极简主义**
   - 图形简洁，易于识别
   - 无多余装饰，突出核心概念

2. **颜色统一**
   - 全橙色系，视觉一致
   - 渐变透明度增加层次感

3. **功能传达**
   - 左右对比直观展示产品价值
   - 箭头清晰表达 AI 处理

4. **品牌识别**
   - 独特的碎片→有序概念
   - 在众多图标中脱颖而出
