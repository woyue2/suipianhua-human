# 新增功能说明文档

## ✅ 回收站功能

### 功能说明
删除的文档不会立即永久删除，而是移到回收站，可以随时恢复。

### 使用方法

#### 1. 删除文档到回收站
- 悬停在文档上
- 点击右侧的垃圾桶图标
- 文档会被移到回收站（不是永久删除）

#### 2. 查看回收站
- 点击侧边栏底部的"回收站"按钮
- 显示所有已删除的文档
- 文档标题会显示删除线

#### 3. 恢复文档
- 在回收站视图中
- 悬停在要恢复的文档上
- 点击绿色的恢复图标（↻）
- 文档会恢复到文档列表

#### 4. 永久删除
- 在回收站视图中
- 悬停在要删除的文档上
- 点击红色的 X 图标
- 确认后文档将被永久删除（无法恢复）

#### 5. 清空回收站
- 在回收站视图中
- 点击右上角的"清空"按钮
- 确认后所有回收站中的文档将被永久删除

### 控制台日志
```
🗑️ Moved to trash: 文档名称
♻️ Restored from trash: 文档名称
❌ Permanently deleted: 文档名称
🗑️ Trash emptied
```

## ✅ 右上角三个图标功能

### 1. 全屏模式 (Maximize2 图标)

**功能**：进入/退出全屏模式

**使用方法**：
- 点击全屏图标
- 或按 F11 键
- 再次点击或按 F11 退出

**状态指示**：
- 激活时图标变为蓝色（primary 色）
- 未激活时为灰色

**控制台日志**：
```
🖥️ Entered fullscreen mode
🖥️ Exited fullscreen mode
```

---

### 2. 演示模式 (PlayCircle 图标)

**功能**：切换演示模式，适合展示和演讲

**使用方法**：
- 点击播放图标
- 弹出提示说明演示模式功能
- 再次点击退出演示模式

**演示模式特点**：
- 隐藏编辑按钮
- 只读模式
- 适合展示和演讲
- 更简洁的界面

**状态指示**：
- 激活时图标变为蓝色（primary 色）
- 未激活时为灰色

**控制台日志**：
```
🎬 Presentation mode: ON
🎬 Presentation mode: OFF
```

---

### 3. 文本格式菜单 (Type 图标)

**功能**：打开文本格式化菜单

**使用方法**：
- 点击文字图标
- 显示格式化下拉菜单
- 点击外部关闭菜单

**可用格式**：
- **粗体** (B)
- *斜体* (I)
- <u>下划线</u> (U)
- 标题 1（大号粗体）
- 标题 2（中号粗体）
- 正文（普通文本）

**状态指示**：
- 菜单打开时图标变为蓝色并高亮
- 菜单关闭时为灰色

**控制台日志**：
```
📝 Format menu: OPEN
📝 Format menu: CLOSE
```

## 💡 使用建议

1. **回收站**：删除文档前不用担心，可以随时从回收站恢复
2. **全屏模式**：编辑长文档时使用，获得更大的工作空间
3. **演示模式**：向他人展示大纲时使用，界面更简洁
4. **格式菜单**：快速访问常用的文本格式选项

---

## ⚠️ 注意事项

1. 回收站中的文档会一直保留，直到手动删除或清空
2. 永久删除操作无法撤销，请谨慎操作
3. 全屏模式可能在某些浏览器中需要用户手势触发
4. 演示模式目前主要是状态切换，具体的只读逻辑可以后续完善
5. 文本格式菜单目前是 UI 展示，具体格式应用功能可以后续实现
